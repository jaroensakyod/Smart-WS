# 🚀 แผนการพัฒนา (Implementation Plan): โปรแกรมสร้างใบงานสำหรับตลาด Teachers Pay Teachers (TpT)

เป้าหมาย: พัฒนา Web App / Extension แบบ Drag & Drop ที่ช่วยให้ครูสร้างใบงานคุณภาพสูง (High-Quality Worksheets) ที่ตรงตามมาตรฐานและข้อกำหนดของการนำไปขายบนแพลตฟอร์ม Teachers Pay Teachers (TpT)

---

## 📍 Phase 1: Core Foundation & Layout (วางรากฐานและระบบจัดวาง)
*เป้าหมาย: ทำให้โปรแกรมสามารถตั้งค่าหน้ากระดาษมาตรฐานและจัดวางองค์ประกอบได้แม่นยำ*

- [x] **1.1 Page Size & Margins (ขนาดกระดาษและขอบ)**
  - ตั้งค่า Canvas (Fabric.js) ให้รองรับสัดส่วน **US Letter (8.5" x 11")** ซึ่งเป็นมาตรฐานหลักของ TpT และ **A4**
  - เพิ่มเส้นขอบจำลอง (Printable Safe Area / Margins) เช่น ขอบ 0.5 นิ้ว เพื่อป้องกันผู้ใช้วางเนื้อหาตกขอบเวลาพิมพ์
- [x] **1.2 Snap to Grid & Guidelines (ระบบจัดระเบียบ)**
  - เปิดใช้งานฟีเจอร์ Snap to Grid ใน Fabric.js
  - สร้าง Smart Guidelines (เส้นสีแดงที่โผล่มาเวลาลากวัตถุไปตรงกับกึ่งกลางหน้ากระดาษ หรือตรงกับวัตถุอื่น)
- [x] **1.3 Alignment Tools (เครื่องมือจัดเรียง)**
  - สร้างปุ่ม UI สำหรับจัดเรียงวัตถุที่เลือก (Align Left, Center, Right, Top, Bottom)
  - สร้างปุ่ม Distribute (กระจายช่องไฟแนวตั้ง/แนวนอนให้เท่ากัน) สำหรับทำช้อยส์ ก. ข. ค. ง.

---

## 📍 Phase 2: Typography & Assets (ระบบข้อความและของตกแต่ง)
*เป้าหมาย: เพิ่มความน่ารักและเครื่องมือที่จำเป็นสำหรับใบงานเด็ก*

- [x] **2.1 Font Management (ระบบจัดการฟอนต์)**
  - เชื่อมต่อ Google Fonts โดยคัดเลือกฟอนต์ที่อ่านง่ายสำหรับเด็ก (เช่น Comic Neue, Fredoka, Quicksand, Mali สำหรับภาษาไทย)
  - **[สำคัญ]** ระบบอัปโหลดฟอนต์ (Custom Fonts) ให้ครูอัปโหลดไฟล์ `.ttf` หรือ `.otf` ของตัวเองได้ (ครูใน TpT นิยมซื้อฟอนต์น่ารักๆ มาใช้เฉพาะ)
- [x] **2.2 Tracing & Writing Lines (เส้นบรรทัดและรอยประ)**
  - เพิ่มฟอนต์แบบรอยประ (Dotted/Tracing Fonts) สำหรับใบงานคัดลายมือ
  - สร้างเครื่องมือวาด "เส้นบรรทัดคัดลายมือ" (เส้นทึบบน-ล่าง, เส้นประตรงกลาง) แบบปรับความกว้างได้
- [x] **2.3 Asset Library (คลังของตกแต่ง)**
  - สร้าง UI Panel สำหรับลากวาง Clipart และ Borders
  - เตรียมชุด SVG พื้นฐาน: กรอบใบงาน (Doodle Borders), กล่องข้อความน่ารักๆ, รูปทรงเรขาคณิต, ลูกศร (สำหรับใบงานโยงเส้น)

## 📍 Phase 3: Worksheet Specific Features (ฟีเจอร์เฉพาะทางสำหรับใบงาน)
*เป้าหมาย: ลดเวลาการทำงานของครู ทำให้สร้างใบงานได้เร็วขึ้น*

- [x] **3.1 Answer Key Generator (ระบบสร้างเฉลย)**
  - เพิ่มปุ่ม "Duplicate as Answer Key"
  - ให้ผู้ใช้สามารถกำหนดวัตถุบางชิ้น (เช่น ข้อความสีแดง) ให้แสดงเฉพาะในโหมด "เฉลย" (Answer Key Mode) และซ่อนในโหมด "โจทย์" (Student Mode)
- [x] **3.2 Auto-Numbering (รันเลขข้ออัตโนมัติ)**
  - เครื่องมือสร้าง "ข้อ" ที่เมื่อกดเพิ่ม จะรันเลข 1., 2., 3. ให้อัตโนมัติ
- [x] **3.3 Page Management (ระบบจัดการหน้า)**
  - รองรับการทำใบงานหลายหน้า (Multi-page) ในโปรเจกต์เดียว (เพิ่มหน้า, ลบหน้า, สลับหน้า)

---

## 📍 Phase 4: Commercial Export & TpT Requirements (ระบบส่งออกเพื่อการค้า)
*เป้าหมาย: ส่งออกไฟล์ให้ได้คุณภาพระดับสิ่งพิมพ์และป้องกันการละเมิดลิขสิทธิ์ (สำคัญมากสำหรับ TpT)*

- [x] **4.1 High-Resolution PDF Export (ส่งออกความละเอียดสูง)**
  - ปรับแต่ง `jspdf` และ `fabric.js` ให้ส่งออก PDF ที่ความละเอียดเทียบเท่า **300 DPI** (ภาพต้องไม่แตกเมื่อพิมพ์)
- [x] **4.2 Flatten PDF (รวมเลเยอร์ป้องกันการก๊อปปี้)**
  - **[ข้อบังคับของ TpT]** ไฟล์ PDF ที่ขายต้องถูก Flatten (แปลงทุกอย่างเป็นภาพแผ่นเดียวก่อนใส่ลง PDF) เพื่อป้องกันไม่ให้ผู้ซื้อใช้โปรแกรมดึง Clipart ออกไปใช้ฟรีๆ ซึ่งผิดกฎลิขสิทธิ์ของนักวาด
- [x] **4.3 Watermarked Preview Generator (สร้างภาพตัวอย่างติดลายน้ำ)**
  - สร้างปุ่ม "Export Preview" ที่จะส่งออกใบงานเป็นไฟล์ JPG/PNG ความละเอียดต่ำ พร้อมประทับคำว่า "PREVIEW" พาดทับหน้ากระดาษ เพื่อให้ครูนำไปใช้เป็นรูปภาพสินค้าในร้าน TpT ได้ทันที

## 📍 Phase 5: Advanced & Future Features (คำแนะนำเสริมและอนาคต)
*เป้าหมาย: เพิ่มมูลค่าให้โปรแกรมและสร้างความแตกต่างจากคู่แข่ง (เช่น Canva)*

- [x] **5.1 Save/Load Project (ระบบบันทึกโปรเจกต์)**
  - แปลง Canvas เป็น JSON (`canvas.toJSON()`) เพื่อให้ผู้ใช้บันทึกไฟล์โปรเจกต์ (เช่น `.smartws`) เก็บไว้ในเครื่อง และโหลดกลับมาแก้ไขได้ในอนาคต
- [x] **5.2 Math Equation Support (สมการคณิตศาสตร์)**
  - พัฒนาระบบ `equation.js` (ที่มีอยู่แล้ว) ให้ใช้งานง่ายขึ้น ผสาน `KaTeX` เข้ากับ Fabric.js ให้ครูพิมพ์เศษส่วน หรือสมการยากๆ แล้วแปลงเป็นภาพเวกเตอร์ลงบนใบงานได้ทันที
- [x] **5.3 Template Marketplace (ระบบเทมเพลต)**
  - มีเทมเพลตสำเร็จรูปให้เลือกตอนเริ่มโปรเจกต์ เช่น ใบงานจับคู่, ใบงาน Bingo, Flashcards

---

## 💡 คำแนะนำเพิ่มเติมสำหรับการทำโปรแกรมนี้ไปขาย/ให้บริการ
1. **จุดขายที่เหนือกว่า Canva:** Canva ทำใบงานได้ แต่ไม่ได้ออกแบบมาเพื่อ "ครูที่ขายบน TpT" โดยตรง โปรแกรมของคุณควรชูจุดเด่นเรื่อง **Flatten PDF อัตโนมัติ**, **การทำ Answer Key ในปุ่มเดียว**, และ **การสร้าง Preview ติดลายน้ำ** ซึ่งเป็น Pain point ที่ครู TpT ต้องเจอทุกวัน
2. **ลิขสิทธิ์ Clipart:** หากคุณจะแถม Clipart หรือ Borders ไปในโปรแกรม ต้องแน่ใจว่าเป็นแบบ Commercial Use หรือวาดเอง เพื่อไม่ให้ครูที่นำไปใช้ต่อโดนฟ้องลิขสิทธิ์
3. **Monetization (การหารายได้จากโปรแกรม):**
   - *Freemium:* ใช้ฟรีแต่จำกัดฟอนต์/กรอบ ส่งออก PDF มีลายน้ำของโปรแกรมคุณ
   - *Pro Version:* จ่ายรายเดือน/รายปี หรือจ่ายครั้งเดียว เพื่อปลดล็อกฟอนต์, อัปโหลดฟอนต์เองได้, ส่งออก PDF แบบ Flatten คุณภาพสูง และไม่มีลายน้ำของโปรแกรม
