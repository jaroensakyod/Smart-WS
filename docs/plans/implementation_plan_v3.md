# 📐 แผนการดำเนินงาน Smart WS (Phase 3: UI & Advanced Tools)

เอกสารนี้ระบุรายละเอียดทางเทคนิคในการแก้ไขระบบ Smart WS ตามความต้องการ 11 ข้อ เพื่อยกระดับประสบการณ์ผู้ใช้และเพิ่มขีดความสามารถในการสร้างใบงาน

---

## 📅 ระยะที่ 1: การปรับปรุงโครงสร้าง UI และเครื่องมือ (Immediate UI Changes)

### 1.1 การปรับสีแถบเครื่องมือ (Toolbar UI Refresh)
- **ไฟล์ที่แก้ไข**: [style.css](style.css)
- **รายละเอียด**: 
  - เปลี่ยนสี `--surface` (หรือเจาะจงที่ `.toolbar`) ให้เป็นสีน้ำเงินเข้มที่มี Contrast สูง (เช่น `#1e293b` หรือสีโทนน้ำเงินสว่าง) เพื่อให้ไอคอนเครื่องมือที่เคยมืดมองเห็นได้ชัดเจน
  - ปรับสีไอคอนพื้นฐาน (svg fill) ให้โดดเด่นขึ้น

### 1.2 การจัดวางปุ่มค้นหารูปภาพใหม่
- **ไฟล์ที่แก้ไข**: [newtab.html](newtab.html)
- **รายละเอียด**: ย้ายปุ่ม `<button id="toolImageSearch">` ไปไว้ที่ตำแหน่งแรกสุดภายในกลุ่ม `.toolbar-tools` ก่อนปุ่มเลือก (Select Tool)

### 1.3 ยกเลิกการส่งออกเป็น Word
- **ไฟล์ที่แก้ไข**: [newtab.html](newtab.html), [export.js](export.js)
- **รายละเอียด**: 
  - ลบโค้ดปุ่ม `<button id="btnExportDOCX">` ใน HTML
  - ลบ Event Listener และฟังก์ชันการสร้างไฟล์ `.doc` ใน JavaScript เพื่อลดขนาดไฟล์

---

## 📅 ระยะที่ 2: การพัฒนาคลังทรัพยากร (Library Expansion)

### 2.1 ทำความสะอาดหมวดหมู่เดิม
- **ไฟล์ที่แก้ไข**: [panel.js](panel.js), [data/svgLibrary.js](data/svgLibrary.js)
- **รายละเอียด**: 
  - ลบหมวด `arrows` และ `animals` ออกจากลิสต์ `CATEGORIES` ใน `panel.js`
  - นำข้อมูล SVG ชุดเก่าที่เกี่ยวข้องออกเพื่อเพิ่มพื้นที่ตัวเลือก

### 2.2 เพิ่มรูปเรขาคณิต (Geometry) สำหรับคณิตศาสตร์พื้นฐาน
- **ไฟล์ที่แก้ไข**: [data/svgLibrary.js](data/svgLibrary.js)
- **รายละเอียด**: เพิ่มรูปทรง 3D/2D ที่จำเป็น:
  - ทรงกรวย (Cone), ทรงกระบอก (Cylinder) แบบละเอียด, ทรงกลม (Sphere) พร้อมเส้นละติจูด
  - พีระมิดฐานต่างๆ (Pyramids), ปริซึม (Prisms)
  - รูปหลายเหลี่ยมสม่ำเสมอ (Pentagon, Hexagon, Octagon)

### 2.3 เพิ่มสัญลักษณ์ (Symbols) และกรอบ (Frames) ให้หลากหลายเหมือน Word
- **ไฟล์ที่แก้ไข**: [data/svgLibrary.js](data/svgLibrary.js)
- **รายละเอียด**: 
  - **สัญลักษณ์**: เครื่องหมายถูก/ผิด, ดาว (4-12 แฉก), หัวใจ, สัญลักษณ์แจ้งเตือน, ไอคอนอุปกรณ์สื่อสาร
  - **กรอบ**: กรอบดอกไม้, กรอบลายเส้นไทยแบบประยุกต์, กรอบแบบมีเงา, และกรอบมุมแบบหยัก

## 📅 ระยะที่ 3: ยกระดับระบบค้นหารูปภาพ (Image Search Upgrade)

### 3.1 ระบบโหลดรูปเพิ่มเมื่อเลื่อน (Infinite Scroll)
- **ไฟล์ที่แก้ไข**: [webImages.js](webImages.js), [newtab.html](newtab.html)
- **รายละเอียด**: 
  - เพิ่มตัวแปร `currentOffset` ในการเรียก API ของ Wikimedia
  - เพิ่ม Scroll Listener ที่ `#imgSearchGrid`: เมื่อเลื่อนถึงด้านล่าง ให้เรียก `doSearch()` ซ้ำโดยเพิ่มค่า `offset`

### 3.2 ปรับขนาดรูปตัวอย่างให้ใหญ่ขึ้น
- **ไฟล์ที่แก้ไข**: [style.css](style.css)
- **รายละเอียด**: 
  - ปรับ `.img-grid` ให้เป็น `grid-template-columns: repeat(2, 1fr)` (แสดง 2 รูปต่อแถว)
  - ขยายขนาด `.img-card` และรูป `img` ด้านในให้ชัดเจนขึ้น

### 3.3 เพิ่มตัวกรอง (Advanced Filters)
- **ไฟล์ที่แก้ไข**: [newtab.html](newtab.html), [webImages.js](webImages.js)
- **รายละเอียด**: 
  - เพิ่ม Checkbox/Chip สำหรับ: `Monoline`, `SVG Only`, `Vector Only`
  - ปรับปรุงการสร้าง Query String ใน `webImages.js` ให้รองรับการกรอง `filetype:svg` หรือคีย์เวิร์ดที่เกี่ยวข้อง

## 📅 ระยะที่ 4: การวาดเส้นขั้นสูง (Advanced Line Drawing)

### 4.1 พัฒนาระบบเส้นมีหัวลูกศร (Arrow Systems)
- **ไฟล์ที่แก้ไข**: [app.js](app.js), [toolbar.js](toolbar.js)
- **รายละเอียด**: 
  - เพิ่มตัวเลือกลักษณะเส้น (Line Settings) ในแผงควบคุม
  - เมื่อผู้ใช้วาดเส้นในโหมด `line`: 
    - สร้างหัวลูกศร (Triangle) ไว้ที่ปลายเส้นอัตโนมัติหากเลือกโหมดลูกศร
    - รองรับเส้นประ โดยใช้พร็อพเพอร์ตี้ `strokeDashArray` ของ Fabric.js

### 4.2 เครื่องมือวาดพื้นฐานเพิ่มเติมเหมือน Word
- **ไฟล์ที่แก้ไข**: [app.js](app.js), [newtab.html](newtab.html)
- **รายละเอียด**: เพิ่มเครื่องมือวาด:
  - เส้นลูกศรสองหัว
  - เส้นโค้ง (Curve/Arc)
  - กล่องข้อความแบบมีลูกศรชี้ (Callout)
