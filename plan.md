# แผนการพัฒนา Smart WS (Smart Worksheet) ระยะที่ 2

ตามความต้องการที่ระบุมา เราจะทำการอัปเกรดระบบเพื่อรองรับการสร้างใบงานที่ซับซ้อนและสวยงามขึ้นในสไตล์ **Monoline (เส้นเดี่ยว ความหนาสม่ำเสมอ)** โดยมีแผนการดังนี้:

## 1. การเพิ่ม Graphic Elements สไตล์ Monoline (ข้อ 1-5, 8)
เราจะปรับปรุง `data/svgLibrary.js` โดยเพิ่มหมวดหมู่ใหม่ๆ และเปลี่ยนรูปแบบกราฟิกให้เป็นเส้นเดี่ยว (Stroke-based) ที่ดูสะอาดตากว่าจาก Wikimedia ทั่วไป:
- **หมวดหมู่ที่เพิ่มใหม่**: 
  - `frames_monoline`: กรอบเส้นเดี่ยวหลากหลายลาย (Minimalistic)
  - `arrows_monoline`: หัวลูกศรแบบต่างๆ, เส้นปะ, เส้นหยัก
  - `symbols_monoline`: วงกลม/สามเหลี่ยมสำหรับกรอกคำตอบ
  - `geometry_monoline`: รูปทรงเรขาคณิต 2D/3D แนวเส้น
  - `animals_monoline`: กราฟิกสัตว์สไตล์การ์ตูนเส้นเดียว (Minimalist Animals)
- **การปรับปรุงการแสดงผล**: เพิ่มตัวเลือก Filter ในแผงด้านซ้ายเพื่อให้เลือกดูเฉพาะกราฟิก "Monoline" ได้ง่ายขึ้น

## 2. ระบบหลายหน้า (Multi-page Support) (ข้อ 6)
ปัจจุบันโปรแกรมรองรับเพียงหน้าเดียว เราจะเปลี่ยนโครงสร้าง `app.js` และ `newtab.html`:
- **UI**: เพิ่มปุ่ม "➕ เพิ่มหน้า" ด้านล่างของพื้นที่ทำงาน
- **Logic**: 
  - จัดการ Canvas ในรูปแบบ Array (`pages[]`) 
  - สามารถสลับหน้า (Active Page) เพื่อแสดงผลและแก้ไขเฉพาะหน้าที่เลือก
  - การจัดการ Scroll ให้ดูต่อเนื่องเหมือนทำงานในโปรแกรมจัดการเอกสาร

## 3. การส่งออกเป็นไฟล์ Word (.docx) (ข้อ 7)
- เพิ่มฟังก์ชันใน `export.js` โดยใช้ไลบรารีภายนอก (เช่น `docx.js`)
- **วิธีการทำงาน**: แปลง Content ของแต่ละ Page บน Canvas ให้เป็นรูปภาพความละเอียดสูง แล้วนำไปวางบนหน้ากระดาษในเอกสาร Word แยกหน้ากันตามลำดับ

## 4. ปรับปรุงการค้นหารูปภาพ (ข้อ 8)
- ปรับแต่ง `webImages.js` ให้เพิ่มพารามิเตอร์การค้นหาที่เน้น "Line Art" หรือ "Vector ClipArt" เพื่อให้ได้รูปที่ใกล้เคียงสไตล์ Monoline มากขึ้น (แทนที่จะเป็นภาพถ่ายทั่วไป)
- เพิ่มหมวดหมู่ "Monoline Library" ไว้เป็นตัวเลือกแรกในแถบด้านซ้าย เพื่อลดการพึ่งพารูปจาก Wikimedia ที่อาจจะไม่ตรงสไตล์
